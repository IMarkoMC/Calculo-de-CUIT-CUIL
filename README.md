# Calculo-de-CUIT-CUIL

Un proyecto viejo, arrancado en la escuela con unos amigos.
La consigna era crear un programa en el cual vos ingresas el DNI y este te da tu CUIT/CUIL.

Los calculos necesarios fueron extraidos de documentacion proporcionada por los profesores, este "script" no funciona bien pero fue una prueba de concepto para hacer la version que esta para celulares basado en Flutter (Dart)
